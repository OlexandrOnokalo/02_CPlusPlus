Завдання 5.
Створити проект, у якому:
Створити три файли: function.h, function.cpp, main.cpp.

У файлі function.срр необхідно написати такі функції для роботи з масивом даних:
	функцію для заповнення масиву випадковими значеннями (або з клавіатури);
	функцію для виведення значень масиву на консоль;
	функцію для пошуку максимального елемента;
	функцію для редагування значення масиву.
Дані функції необхідно написати для роботи з масивом цілих, дійсних і символьних значень. (Для кожного типу написати окрему функцію).

У файлі function.h потрібно:
	описати прототипи всіх функцій;
	зробити перевірку, чи визначена константа, що вказує на тип даних (#ifdef INTEGER);
	створити узагальнюючі імена функції, якими буде визначено функцію відповідного типу залежно від константи, 
	зазначеної у файлі main.cpp (#define show ShowInt).

У файлі main.cpp потрібно :
	у функції main викликати всі функції через переозначені (узагальнені) імена з файлу function.h (show( ));
	визначити константу, яка вказує який тип даних буде використовуватися. 
		Наприклад INTEGER — цілі дані, CHAR — символьні, DOUBLE — справжні (#define INTEGER).

Ім'я функціям потрібно давати так, щоб вони відповідали своїй суті. 
Наприклад, функція для виведення значень масиву цілих чисел на консоль — ShowArrayInt(). 
Коли перевизначаємо функцію за допомогою #define, то потрібно давати загальне ім'я. 
Наприклад, #define ShowArray ShowArrayInt.
